QUICK STEPS TO FINISH
=====================

1. PUSH CODE TO GITHUB (5 min)
------------------------------
Run these commands in terminal:

cd /Users/aisaaq/Downloads/appsight-ai_-app-competitor-analysis

git init
git add .
git commit -m "Initial commit"
git branch -M main

# Create new repo on GitHub.com first, then:
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git push -u origin main

(Replace YOUR_USERNAME and YOUR_REPO_NAME with your actual GitHub username and repo name)


2. DEPLOY TO RENDER (10 min)
-----------------------------
1. Go to https://render.com
2. Click "Deploy a Web Service"
3. Click "Connect GitHub" → Select your repo
4. Settings:
   - Name: betterapp-backend
   - Root Directory: server
   - Build: npm install
   - Start: npm start
   - Instance: Free
5. Add Environment Variables (click "Advanced"):
   Copy all from server/.env file
6. Click "Create Web Service"
7. Wait 5-10 min → Get your URL!


3. REBUILD DESKTOP APP (2 min)
-------------------------------
cd electron
npm run build:mac

Done! New .dmg will have your logo!
