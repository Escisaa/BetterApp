  BetterApp - Quick Install
═══════════════════════════════════════════════════════════════

1. Drag BetterApp to Applications folder
2. Open Terminal and run:
   xattr -cr /Applications/BetterApp.app
3. Open BetterApp from Applications

That's it! The app will open without warnings.
