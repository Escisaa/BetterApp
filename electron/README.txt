  BetterApp - Installation Instructions
═══════════════════════════════════════════════════════════════

STEP 1: Open the DMG
  • Double-click "BetterApp.dmg" to mount it

STEP 2: Install BetterApp
  • Drag "BetterApp" to the "Applications" folder

STEP 3: Open BetterApp (IMPORTANT!)
  macOS may show a security warning. Here's how to open it:

  OPTION A (Recommended):
  1. Open Finder
  2. Go to Applications folder
  3. Right-click (or Control-click) "BetterApp"
  4. Select "Open" from the menu
  5. Click "Open" in the security dialog
  6. The app will now open normally

  OPTION B (Alternative):
  1. Open System Settings > Privacy & Security
  2. Scroll down to find "BetterApp was blocked..."
  3. Click "Open Anyway"
  4. Click "Open" in the confirmation dialog

STEP 4: First Launch
  After opening once, BetterApp will work normally.
  You can double-click it like any other app.
